# mqtt-robot-nodemcu

 * Upload this to NodeMCU board with MicroPython.

 * Change `config.py`

 * Connect to WiFi  

       import network
       wifi = network.WLAN()
       wifi.connect('robot', '12345678')

 * Reboot the board

Web part is available at https://github.com/VerosK/mqtt-robot-webdriver/
